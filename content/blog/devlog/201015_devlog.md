## 2. 배운점

- redux 구조 명확히 파악
  - view -> action -> dispatcher -> store
  - reducer : state와 action을 입력받아서 새로운 state를 반환하는 함수
  - action : type, payload로 이루어진 객체
  - action creator : action을 생성하는 함수
  - dispatch : action을 store에 보내는 함수
  - store : 상태값이 갖는 객체
